polymask.js

jQuery Plugin for Responsive Polygon Masking and Clipping of HTML elements

View on GitHub Download .zip Download .tar.gz

Welcome to jQuery polyMask Plugin

This jQuery plugin makes polygon clipping for your HTML elements easy. It handles crossbrowser compatibility. And it is responsive!

Demo

Click here for demo

How To Use

Add the plugin to the head or your document <script src="https://code.jquery.com/jquery-1.11.3.min.js"></script> <script src="jquery.polymask.js></script>

*Responsive Clipping <div class="masked" data-mask-path="50% 0%, 100% 50%, 50% 100%, 0% 50%"> <img src="http://lorempixel.com/g/800/800/" /> </div>
